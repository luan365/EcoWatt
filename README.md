coWatt

Aplicação React para monitoramento e análise de consumo de energia, com sugestões para eficiência energética.

Tecnologias

React + TypeScript (Create React App)

Pré-requisitos

Node.js (versão 16+ recomendada)

npm (ou yarn)

Instalação git clone https://github.com/luan365/EcoWatt.git cd EcoWatt npm install

*npm run build*

*serve dist* para testar localmente
